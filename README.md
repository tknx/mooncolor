# mooncolor

### An attempt at accurate mooncolor simulator for Hue bulbs

*Warning: This is 100% AI-coded between Claude Opus 4.7 and Gemini Pro. It is for a fun little thing so I did not sweat it. User beware!*

I 3d printed a moon and put a Hue bulb in it.

And it was good.

Time went by. And suddenly I had a thought...

And I dumped that thought into an AI prompt and let it run wild.

And it was good.

So here we are... A small little program that should accurately model the color of the moon where you are using a Hue bulb. I am including a profile for the model bulb I am using, so if you are using a different one, you may need to figure out the color-space mapping.

The program itself is documented... so godspeed. There is no support here.
